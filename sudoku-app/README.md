# Sudoku Premium

Un juego de Sudoku moderno, limpio y profesional desarrollado en React. 
Diseñado para ofrecer una experiencia relajante con funcionalidades avanzadas de accesibilidad y soporte tanto para móvil como para escritorio.

## Características Principales

*   **Modos de Juego Flexibles:** 
    *   **Clásico (9x9):** El formato tradicional con números del 1 al 9.
    *   **Hexadecimal (16x16):** Un desafío mayor con valores del 0 a la F.
*   **Dificultades Adaptativas:** Fácil, Medio y Difícil, que ajustan la cantidad de números pre-rellenados y el número de vidas (errores permitidos) del jugador.
*   **Sistema de Vidas y Puntajes:** Castiga los errores y recompensa la velocidad y precisión mediante un sistema de puntaje y un Leaderboard.
*   **Auto-guardado (Persistencia Local):** Si cierras la pestaña por accidente, tu partida (tablero, tiempo y vidas) se guarda automáticamente en el navegador y puedes retomarla cuando quieras.
*   **Sistema Anti-Trampas (Pausa):** Botón de pausa que oculta el tablero mediante un elegante efecto de desenfoque (`blur`) y detiene el temporizador, evitando que el usuario piense jugadas gratis.
*   **Auto-limpieza de Notas:** Cuando aciertas un número, el juego limpia automáticamente las notas coincidentes en la misma fila, columna y cuadrícula, mejorando drásticamente la calidad de vida del usuario.
*   **Soporte Multidispositivo (Numpad):** Incluye un teclado numérico táctil dinámico (Numpad) que se activa únicamente en dispositivos móviles o tablets, asegurando la mejor experiencia independientemente del hardware.
*   **Tema Claro/Oscuro:** Adaptable a las preferencias del sistema del usuario o manualmente ajustable desde el menú.

## Tecnologías

*   **[React](https://react.dev/):** Biblioteca principal de UI.
*   **[Vite](https://vitejs.dev/):** Bundler y entorno de desarrollo ultra-rápido.
*   **[Framer Motion](https://www.framer.com/motion/):** Para transiciones de interfaz, micro-interacciones y animaciones fluidas (modales, botones, corazones).
*   **[Lucide React](https://lucide.dev/):** Set de iconos vectoriales modernos y ligeros.
